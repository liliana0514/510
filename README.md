## How to Run
Put the following In your .gitignore file
```
venv
*.pyc
```

Put the following in your requirements.txt file
```
# requirements.txt
streamlit
```

Open the terminal and run the following commands:
```
python -m venv venv             
source venv/Scripts/activate        
pip install -r requirements.txt 
```
